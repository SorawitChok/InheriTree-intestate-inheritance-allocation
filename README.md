# InheriTree: Streamlining Intestate Inheritance Allocation under Thai Civil Law by Leveraging NLP-Powered Chatbot and Web Application

This repository contains the data and resources as describe in the **Data Availability** section of the paper titled **"InheriTree: Streamlining Intestate Inheritance Allocation under Thai Civil Law by Leveraging NLP-Powered Chatbot and Web Application."** These resources are made publicly available to facilitate further research and development in the field.

## Abstract
The process of allocating the intestate inheritance among the statutory heirs is sophisticated yet occurs regularly. Many scholars have attempted to develop automated allocation systems to tackle this challenging task. However, most amply existing systems rely on conventional form-based input, which may overwhelm the general users. Furthermore, no existing system concerning intestate inheritance allocation according to the Civil and Commercial Code of Thailand is publicly available. We proposed "InheriTree", a user-centric system for navigating the complexities of inheritance rights per the Civil and Commercial Code of Thailand by leveraging the potential of the natural language processing (NLP)-powered chatbot along with an interactive web application. Diverse NLP techniques coalesce into InheriTree's chatbot system to ensure that the chatbot can handle various structures of input text from the user. This chatbot is seamlessly integrated with the Facebook Messenger platform, which bestows the user with a familiar and easily accessible chat interface. The information derived during the chat session then passes through our web application to construct and visualize a modifiable family tree accordingly. The core function of InheriTree is the precise allocation of the inheritance ratio employing the tree-based "three-step inheritance allocation algorithm", adept at managing intricate inheritance scenarios. This proposed algorithm also enables our system to handle the case with the heir according to the right of representation, which is unattainable for any existing system. The proposed system's inheritance allocation results achieved 100% accuracy from end-to-end testing across 100 test scenarios.
